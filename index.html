macOS (Gatekeeper): These ZIPs are not signed with an Apple Developer ID, so macOS may quarantine them after download. That is normal for independent builds. After unzipping, open Terminal, cd to the folder that contains the .app bundle, then run the xattr -cr command shown on each row below (it removes quarantine for that bundle). Only run it for software you trust. If the app is not in the current directory, pass a full path instead (for example xattr -cr ~/Downloads/MyGame.app).
